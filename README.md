# Computer Science Notes

This repository is a collection of computer science notes, generators, and study resources.

## Topics

- `api/` - API notes, API course material, and related generators.
- `sql/` - SQL notes and database practice material.
- `machine-learning/` - Machine learning notes, examples, and experiments.
- `data-structures-and-algorithms/` - DSA notes and coding practice.

### API Course DOCX Generator

Location:

```text
api/api-course-docx-generator/
```

This project generates a polished `.docx` file for API course notes using Node.js and the `docx` package.

Run it from its folder:

```bash
cd api/api-course-docx-generator
npm install
node api_course.js
```
